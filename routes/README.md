# CoolEmails

Buy and Send Emails

## Prerequisites
> Node.js 10.7.0 or latest
> Npm 6.1.0 or latest

## Getting Started
Download the repository
```
git clone git@github.com:rojasleon/server.git
```
or Download the ZIP

### mode={development}
Install
```
npm install
```
Run
```
npm run dev
```
